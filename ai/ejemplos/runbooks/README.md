## Proposito
Esta carpeta documenta como leer ejemplos de runbooks sin confundirlos con los runbooks operativos oficiales.

## Regla
- Los runbooks ejecutables del framework viven en `../../runbooks/`.
- Esta carpeta solo debe contener ejemplos, notas o variantes didacticas.
- No copies un ejemplo de esta carpeta como contrato operativo de un proyecto real.

## Runbook oficial relacionado
- [crear-proyecto-real-con-agente.md](../../runbooks/crear-proyecto-real-con-agente.md)

## Uso recomendado
1. Lee primero el runbook oficial.
2. Usa ejemplos solo para entender tono, nivel de detalle o forma de pedir confirmaciones.
3. En proyectos reales, adapta el runbook al dominio, stack, gates y rutas efectivas.

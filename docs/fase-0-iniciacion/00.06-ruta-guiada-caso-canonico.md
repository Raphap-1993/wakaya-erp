<a id="historia-del-mvp-canonico"></a>
## Historia del MVP canonico
El caso canonico representa un sistema de gestion de expedientes para una entidad con operacion on-premise, integracion con IAM corporativo y auditoria obligatoria.

El MVP de referencia se centra en tres historias:
- `HU-02` consulta por bandeja de trabajo
- `HU-03` cambio de estado con validacion
- `HU-04` historial de auditoria visible

Estas historias se materializan como:
- `001-bandeja-trabajo-expedientes`
- `002-cambio-estado-expediente`
- `003-historial-auditoria-expediente`

## Estilo de arquitectura
Wakaya ERP usa frontend Angular desacoplado y backend Quarkus modular. El MVP prioriza baja complejidad operativa, seguridad OIDC/RBAC y trazabilidad de cambios.

## Frontend
- Aplicacion Angular 21 en workspace Nx.
- App principal: wakaya-erp-web.
- Consumo de API /api/reservations.

## Backend
- Servicio Quarkus: wakaya-erp-api.
- Paquete Java: com.wakaya.erp.
- API REST para reservations, salud y seguridad.

## Base de datos
- Base principal: wakaya_erp.
- Migraciones con Flyway.
- Auditoria de cambios de estado como entidad de dominio.

## Seguridad
- OIDC issuer: https://iam.wakayaecolodge.com/realms/wakaya-erp.
- Audience: https://wakayaecolodge.com/api.
- Claim de roles: roles.
- Permisos: reservation:read, reservation:write, reservation:approve.

## Observabilidad
- Prometheus: https://prometheus.wakayaecolodge.com.
- Grafana: https://grafana.wakayaecolodge.com.
- Logs con redaccion de PII.

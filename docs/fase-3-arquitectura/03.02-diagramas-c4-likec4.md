Este proyecto usa LikeC4 como fuente unica para los diagramas C4.

## Diagramas obligatorios
- Fase 3 Arquitectura: contexto y contenedores.
- Fase 3 Arquitectura: deployment cuando la estrategia de despliegue ya este definida.

## Diagramas opcionales
- Componentes cuando un modulo o servicio tenga suficiente complejidad para justificar mayor detalle.
- Vistas adicionales por dominio, entorno o integracion si ayudan a explicar decisiones relevantes.

## Regla de consistencia
- `docs/fase-3-arquitectura/03.00-arquitectura.md` describe la arquitectura.
- `docs/fase-3-arquitectura/03.01-decisiones-tecnologia.md` justifica la tecnologia.
- `docs/fase-3-arquitectura/03.03-plan-despliegue.md` define ambientes y release.
- `likec4/` refleja visualmente esas decisiones y debe mantenerse sincronizado.

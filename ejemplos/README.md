# Ejemplos de referencia

[README principal](../README.md) | [Indice docs](../docs/README.md)

Estos ejemplos usan un mismo caso de muestra: un sistema de gestion de expedientes para una entidad con operacion on-premise y necesidad de auditoria.

## Caso canonico
- Dominio: gestion de expedientes.
- Slice MVP de referencia: `HU-02 Consulta por bandeja de trabajo`, `HU-03 Cambio de estado con validacion` y `HU-04 Historial de auditoria`.
- Features end-to-end: `001-bandeja-trabajo-expedientes`, `002-cambio-estado-expediente` y `003-historial-auditoria-expediente`.

# Ejemplo de operacion y metricas

## Objetivo
Operar el sistema con visibilidad de disponibilidad, desempeño y evolución funcional.

## Monitoreo activo
- API disponible.
- Tiempo medio de respuesta de bandeja.
- Latencia de cambio de estado.
- Latencia de consulta de historial.
- Consumo de recursos.
- Estado de integración IAM.

## Alertas y umbrales
- Error 5xx mayor a 3 por ciento.
- Tiempo de respuesta promedio mayor a 2 segundos.
- Uso de CPU mayor a 85 por ciento durante 10 minutos.

## KPIs y SLIs
- Consultas de bandeja por dia.
- Cambios de estado exitosos por dia.
- Consultas de historial por dia.
- Tiempo medio de atencion.
- Tasa de error por operación.
- Disponibilidad mensual.

## Frecuencia de revisión
- Diario para alertas.
- Semanal para KPIs.
- Mensual para backlog evolutivo.

## Backlog evolutivo
- Firma digital.
- Reportes avanzados.
- Integracion con ERP.

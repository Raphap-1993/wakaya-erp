# Operaciones

[README principal](../README.md) | [Indice docs](../docs/README.md)

Carpeta para artefactos operativos del proyecto organizados por fase.

## Regla de uso
- `ops/` contiene artefactos aplicados al proyecto actual.
- Los archivos raiz de cada fase funcionan como baseline operativo inicial del proyecto.
- Los artefactos por feature o release viven en subcarpetas como `features/`.
- Las plantillas reutilizables viven en `plantillas/`, no en `ops/`.

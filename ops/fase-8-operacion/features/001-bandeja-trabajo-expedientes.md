# Operacion feature - Bandeja de trabajo de expedientes

## Monitoreo clave
- latencia p95 de listado
- tasa de errores por consulta
- porcentaje de resultados vacios

## Alertas sugeridas
- alerta si la latencia p95 supera el objetivo acordado
- alerta si aumentan errores `500` en consultas de expediente

## Backlog evolutivo
- exportacion de resultados
- filtros guardados por usuario
- vista de historial completa

# Operacion feature - Historial y auditoria visible de expediente

## Monitoreo clave
- latencia de consulta de historial
- tasa de consultas fallidas
- volumen de eventos visibles por expediente

## Alertas sugeridas
- alerta si la latencia del historial supera el umbral acordado
- alerta si se detectan expedientes con cambios auditados no visibles en historial

## Backlog evolutivo
- exportacion puntual de historial
- comparacion visual entre eventos
- filtros avanzados para auditor

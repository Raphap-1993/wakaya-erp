## Resumen
Esta release lleva la plantilla a un nivel mas didactico y operativo: ahora incluye una ruta guiada de onboarding en una sola lectura y cierra el caso canonico con tres features end-to-end.

## Cambios principales
- ruta guiada del caso canonico para onboarding rapido,
- cierre del trio `HU-02`, `HU-03`, `HU-04`,
- ejemplo canonico mas consistente entre fases,
- nueva feature `003-historial-auditoria-expediente`,
- release formal alineada al nuevo nivel de madurez del template.

## Snapshot del estado actual
- `docs/` incluye flujo general, checklist de adopcion y ruta guiada del caso canonico.
- `ejemplos/` cuenta una sola historia consistente de expedientes a traves de las fases.
- `specs/001`, `specs/002` y `specs/003` cubren lectura, escritura e historial visible.
- `src/`, `tests/`, `qa/` y `ops/` ya muestran aterrizaje real por feature.

## Resumen
Esta release completa el patron de consistencia entre plantillas y ejemplos, termina de orientar el runbook operativo y mantiene sincronizado el versionado documental del template.

## Cambios principales
- guia operativa completada en `plantillas/fase-7-deploy/runbook.md`,
- enlaces bidireccionales `plantillas <-> ejemplos` agregados en fases 0, 1 y 8,
- `ejemplos/fase-0-iniciacion/README.md` alineado con la estructura usada por el resto de ejemplos,
- nota de cierre del recorrido guiado corregida para volver al inicio real en `docs/transversal/90.12-mapa-ia-por-fase.md`,
- `CHANGELOG.md` y `releases/` sincronizados con la nueva version `v10.4.0`.

## Snapshot del estado actual
- `plantillas/` ya ofrece orientacion consistente en artefactos de inicio, analisis, arquitectura, QA, deploy y operacion.
- `ejemplos/` mantiene mejor trazabilidad hacia la plantilla base correspondiente en las fases con mayor uso.
- `docs/` conserva el recorrido guiado completo desde el README raiz hasta el cierre transversal.
- `releases/` vuelve a reflejar las versiones relevantes del template con snapshot documental por iteracion.

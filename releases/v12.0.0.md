# v12.0.0 - Plantilla lista para entornos regulados

[README principal](../README.md) | [Indice docs](../docs/README.md) | [Volver a releases](README.md)

## Resumen
`v12.0.0` cierra 11 brechas detectadas en la revision post-v11 para llevar la plantilla desde una "plantilla AI-first con scaffolding" a una **plantilla enterprise/regulada**: supply-chain firmada, Terraform multi-entorno, Kubernetes progresivo, GitOps, audit trail inmutable, ROPA/DPIA, FinOps y sostenibilidad, golden paths y testing avanzado.

## Destacado
- **Supply-chain verificable**: cosign keyless, SLSA, SARIF en Security tab, politicas Kyverno que solo admiten imagenes firmadas.
- **IaC multi-entorno real**: `envs/{dev,staging,prod}` + modulos reutilizables + state lock en `ops/infra/aws/bootstrap/`.
- **Despliegue progresivo**: canary con AnalysisTemplate Prometheus, PDB, HPA, NetworkPolicy default deny, External Secrets Operator.
- **GitOps**: app-of-apps ArgoCD con auto-sync en dev/staging y manual en prod.
- **Audit trail**: `contracts/events/audit-log.schema.json` + plantillas ROPA y DPIA (GDPR art. 30 y 35).
- **FinOps + sostenibilidad**: `90.22-finops.md`, `90.23-sostenibilidad.md`, tagging obligatorio en Terraform.
- **Testing avanzado**: axe + Playwright, Pact, k6 con thresholds SLO.
- **Observabilidad como codigo**: `ops/observability/` con pipeline OTEL + reglas Prometheus + dashboard Grafana.
- **Golden path**: `scripts/new-service.sh` bootstrap completo de servicio nuevo.

## Snapshot
- 4 stacks (`node-next`, `java-monolith`, `quarkus-angular`, `spring-react`) con `catalog-info.yaml` para Backstage.
- 7 workflows (`docs`, `ci`, `security`, `release`, `pr`, `release-please`, `preview`).
- 6 ADRs (`ADR-001` a `ADR-006`).
- 24 documentos transversales (`90.00` a `90.23`).
- Contratos: OpenAPI 3.1 (con `bearerAuth` + ejemplos), AsyncAPI 3.0, JSON Schemas de eventos + audit log.
- IaC: 4 modulos Terraform, 3 envs, bootstrap, 6 overlays/policies Kubernetes.
- Observabilidad: OTEL collector, Prometheus rules, Grafana dashboard.

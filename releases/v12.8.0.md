# v12.8.0 - Fase UX ejecutable y stack real validado

[README principal](../README.md) | [Indice docs](../docs/README.md) | [Volver a releases](README.md)

## Resumen
`v12.8.0` cierra el siguiente paso del agente interno: despues de crear un proyecto real, ahora puede ejecutar Fase 2 UX sobre ese proyecto y producir salidas utiles para Markdown, Penpot y Angular mock. Tambien deja el stack Quarkus/Angular preparado para compilar, probar y arrancar en un caso real.

## Destacado
- **Nuevo comando `ux`**: `node scripts/ai-framework-agent.mjs ux --project <ruta>` ejecuta el task packet de Fase 2 sobre un proyecto generado.
- **UX Markdown -> Penpot -> Angular mock**: el agente crea flujos, wireframes Markdown, criterios UX, brief Penpot, brief Angular y prompts generados para proveedor IA.
- **Mock Angular conectado**: se genera `frontend/libs/feature-expedientes/` y se actualiza `frontend/apps/web/src/main.ts` para montar el mock de bandeja de expedientes.
- **Correccion de selector raiz**: el componente mock usa `selector: 'app-root'`, compatible con el `index.html` generado.
- **Stack ejecutable**: Quarkus/Angular queda alineado con Java 21, Angular 21, TypeScript 5.9, pruebas headless y datasource H2 dev/test.

# v12.9.0 - Penpot local para prototipos de Fase 2

[README principal](../README.md) | [Indice docs](../docs/README.md) | [Volver a releases](README.md)

## Resumen
`v12.9.0` completa la operacion de Fase 2 UX: el agente no solo genera Markdown, briefs y Angular mock, tambien deja una instancia Penpot local reproducible con Docker para mostrar prototipos clickables.

## Destacado
- **Penpot Docker local**: `ux` genera `ops/prototyping/penpot/` con scripts de arranque, estado y apagado.
- **Runbook operacional**: la documentacion indica como levantar Penpot, crear el archivo de prototipo y registrar evidencia.
- **Gate UX Ready reforzado**: ahora la evidencia incluye URL Penpot, checklist, pantallas minimas y trazabilidad RF/HU/spec -> wireframe -> prototipo -> Angular mock.
- **Caso real ejecutado**: `C:\template\caso-real1\expediente` quedo con Penpot disponible en `http://localhost:9001`.

## Comando validado
```powershell
node scripts/ai-framework-agent.mjs ux --project C:\template\caso-real1\expediente --feature 001-bandeja-trabajo-expedientes
```

Resultado:
- 26 artefactos UX, Angular mock y Penpot generados o actualizados.
- 87 archivos Markdown generados/revisados sin hallazgos desde `ux`.
- 88 archivos Markdown revisados sin hallazgos despues de registrar evidencia operativa.
- Validacion de instanciacion sin hallazgos.

## Operacion Penpot validada
```powershell
powershell -ExecutionPolicy Bypass -File C:\template\caso-real1\expediente\ops\prototyping\penpot\start-penpot.ps1
```

Resultado:
- Compose oficial descargado desde Penpot.
- Contenedores levantados: frontend, backend, exporter, postgres, valkey y mailcatcher.
- `http://127.0.0.1:9001` respondio HTTP 200.

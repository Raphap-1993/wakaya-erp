# Prototype Validation - Wakaya ERP Public Site

[README principal](../../README.md) | [Specs](../README.md)

## Validacion
| Criterio | Estado | Observacion |
|---|---|---|
| La home se siente propia de Wakaya | PENDIENTE | Espera revision del primer HTML navegable. |
| Hospedaje domina la jerarquia publica | PENDIENTE | Validar en Home y Habitaciones. |
| Eventos y Full Day no compiten con hospedaje | PENDIENTE | Revisar peso visual y CTA. |
| El booking bar se ve premium y util | PENDIENTE | Validar integracion con hero humano. |
| El flujo a prereserva se entiende | PENDIENTE | Validar desde hero, card y detalle. |
| El tono visual no se siente generico | PENDIENTE | Revisar contra referencia actual de Wakaya. |
| El prototipo puede abrirse sin build complejo | PENDIENTE | Espera entrega HTML5. |

## Decision
`gate-prototype-ready` y `gate-spdd-approved` siguen pendientes hasta revisar
el prototipo HTML5 navegable con criterio humano.

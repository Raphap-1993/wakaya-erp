# Prototype - Wakaya Bungalow Backoffice Media

[README principal](../../README.md) | [Specs](../README.md)

## Modo
HTML5-first

## Ruta del prototipo
- `specs/004-bungalow-backoffice-media/prototype-html5/index.html`

## Qué debe demostrar
- cabecera sticky compacta
- tabs `Operación`, `Ficha web`, `Textos web`
- portada web con preview y acción de reemplazo
- galería con grid de miniaturas y acciones por activo
- copy ES/EN separado del flujo principal
- jerarquía visual más compacta que la pantalla actual

## Escenario mock
- Bungalow Doble activo
- visible en home
- 4 imágenes de galería
- tarifa PEN 280
- área 32 m2
- contenido ES completo
- contenido EN incompleto

## Alcance del prototipo
- No conecta a API real.
- No sube archivos reales.
- No optimiza imágenes reales.
- Sí valida estructura, densidad, agrupación y semántica de acciones.

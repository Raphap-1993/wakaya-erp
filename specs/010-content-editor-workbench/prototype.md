# Prototipo textual - Wakaya Content Editor Workbench

## Modalidad

Wireframe textual aprobado. El Product Owner pidió expresamente continuar solo
con texto y no mostrar un prototipo en navegador.

## Wireframe canónico

```text
Contenido público > Módulo

Módulo                 Estado · última actualización
                       [Vista previa] [Historial] [Guardar y publicar]

┌ Lista ─────────────┐ ┌ Editor ──────────────────────────┐
│ Elemento 1         │ │ Elemento seleccionado            │
│ Elemento 2         │ │ [ES] [EN]                        │
│ Elemento 3         │ │ Contenido                        │
│ ...                │ │ Imagen                           │
│ Configuración web  │ │ Visibilidad y orden              │
└────────────────────┘ │ ▸ Opciones avanzadas             │
                       └───────────────────────────────────┘
```

Preview e historial aparecen como paneles temporales y no alteran el ancho del
editor principal.

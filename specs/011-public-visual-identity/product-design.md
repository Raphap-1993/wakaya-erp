# Product Design - Identidad visual pública

## Problema

La web pública mezcla verdes, dorados y tipografías genéricas. La marca no se
percibe consistente entre Home, Bungalows, Experiencias y páginas internas.

## Objetivo

Unificar la experiencia pública sin alterar su arquitectura ni sus flujos:

- Lora para títulos editoriales.
- Montserrat para párrafos, navegación, botones y formularios.
- Verde ecolodge `#295331` como color principal.
- Verde lima `#b6ce87` como acento.
- Beige arena `#eee9dd` como fondo y descanso visual.

## Decisión

La referencia visual fue entregada y aprobada por el Product Owner el
2026-07-16. El verde lima se limita a acentos y CTA; para texto sobre fondos
claros se usa una variante accesible `#536c2c`.

## Alcance

Web pública ES/EN, incluidos Home, listado y detalle de bungalows, servicios,
galería y páginas corporativas. El backoffice queda fuera del alcance.

# Spec funcional - Identidad visual pública

## Requerimientos

- RF-011-01: toda página pública ES/EN debe usar Lora en títulos semánticos.
- RF-011-02: textos, controles, navegación y CTA públicos deben usar Montserrat.
- RF-011-03: la web pública debe usar la paleta `#295331`, `#b6ce87` y
  `#eee9dd` mediante tokens reutilizables.
- RF-011-04: el backoffice debe conservar su tipografía y tema operativo.
- RF-011-05: no deben quedar acentos dorados retirados en los estilos públicos.

## Criterios de aceptación

- Las fuentes forman parte del artefacto de build.
- Los títulos mantienen legibilidad responsive.
- CTA y texto de acento conservan contraste suficiente.
- No cambia ningún endpoint, dato persistido ni flujo de reserva.
